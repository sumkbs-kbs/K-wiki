# 이게 무료? 5분만에 내 목소리 복제해서 영상 자동화하기

## 요약
에딕초이 채널에서 공개. Claude Code 구독만으로 무료/자동/프라이버시 안전하게 나만의 목소리로 영상제작 자동화하는 워크플로우를 설명합니다. VoiceBox(로컬 음성 복제) + HyperFrames(AI 영상생성) 두 도구 조합이 핵심.

## 핵심 내용

### Workflow(2단계, 5분 완료)
1. **VoiceBox**: 30초~1분 샘플만으로 내 목소리 복제. 로컬 실행으로 외부 서버 전송 없음(프라이버시 안전). GitHub ★25,000+
2. **HyperFrames**: HeyGen 제작. "이런 영상 만들어줘" → HTML로 자동 생성, 상업용 무료 라이선스, Claude Code에 한 줄 명령으로 스킬 설치

### 결과물
- 한 마디도 녹음 안 하고, 한 줄도 코드 안 짬
- 음성 복제 + 모션 그래픽 모두 AI 자동 생성
- 한국어 한계 존재(윤리적 고려사항 언급)

### 핵심 메시지
- 결국 중요한 건 **디렉터의 안목**
- 차이는 도구 선택이 아닌 디렉터의 안목에서 나옴
