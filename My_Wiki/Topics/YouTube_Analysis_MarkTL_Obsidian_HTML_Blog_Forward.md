# 에딕초이 : 이게 무료? 5분만에 내 목소리 복제해서 영상 자동화하기 | 클로드코드+보이스박스+하이퍼프레임

## 요약
에딕초이 채널에서 Claude Code를 사용해 30초~1분 샘플로 내 목소리를 복제, HyperFrames를 통해 HTML 영상 자동 생성하는 워크플로우를 설명합니다. 외부 서버 전송 없이 로컬 VoiceBox(25kstars)를 사용해 프라이버시 완전안전.

## 핵심 도구
**VoiceBox (로컬 음성복제)**
- 30초~1분 샘플만으로 내 목소리 복제 (openclaw 25K stars)
- 외부 서버 전송 없이 로컬 실행 → 프라이버시 완전 안전
- openclaw 25K+ stars의 신뢰성

**HyperFrames (에이아이 영상자동화)**
- HeyGen 제작, "에이아이 영상 생성 도구"
- "이런 영상만들어줘" → HTML 자동 생성
- 상업용 무료 라이선스
- 클로드코드 한 줄 명령으로 스킬 설치

## 타임라인
00:00 - 오늘 워크플로우(2단계)
00:35 - VoiceBox 소개
01:12 - 클로드코드 설치 시작
    HyperFrames 소개 (에이아이로 영상)
    5분만에 결과물 완성
01:28 - 솔직 후기 (한국어 한계 + 윤리)

## 핵심 인사이트
- **에이아이의 도구 진화:** 음성복제+영상 자동생성 모두 무료/오픈소스로 가능
- **프라이버시 우선:** VoiceBox의 로컬 실행 구조가 핵심 (openclaw 25K stars를 통한 신뢰성)
- **클라우드 vs 로컬 차이:** 외부 서버 전송 X = 개인데이터/음성 절대 유출되지 않음
- **비용 제로:** Claude Code 구독만 있으면 완전 무료 워크플로우
- **에이아이 영상자동화의 미래:** 한 마디도 녹음안 하고 영상 제작이 가능한 시점 도달
