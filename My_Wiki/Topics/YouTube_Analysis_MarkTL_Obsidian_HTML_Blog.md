# 옵시디언 노트가 딸깍 한번에 HTML 블로그로! 카파시 추천한 Codex CLI 활용 플러그인 공개

## 요약
배움의 달인 채널에서 공개. Karpathy가 "AI 출력의 다음 기본값은 HTML"이라고 말한 시대에, Obsidian 마크다운 노트를 클릭 한 번에 인터랙티브 HTML 블로그로 자동 변환하는 무료 플러그인 'MarkTL'을 소개합니다. Codex CLI / Claude CLI 자동 연동, GitHub Pages 무료 배포, Disqus 댓글까지 원클릭.

## 핵심 내용

### Karpathy의 HTML 시대 주장
- "AI 출력의 다음 기본값은 HTML" — Karpathy가 트위터에서 강조한 트렌드
- 마크다운 vs HTML: AI에게 HTML은 "10차선 고속도로"
- 메모 AI 출력의 4단계 진화: 손글씨 → 마크다운 → HTML → 뉴럴링크

### MarkTL 플러그인
- Obsidian 마크다운 노트 → 인터랙티브 HTML 블로그 자동 변환
- **디자인 프리셋 5종**: Shareable / Public / Research / Memo / Interactive
- **Codex CLI / Claude CLI 연동 모드**: 별도 API 비용 0원
- **GitHub Pages 연동**: HTML 파일 생성 + 자동 배포
- **Disqus 댓글** 지원
- **인터랙티브 리포트, 비주얼 아티팩트** 등 HTML 특화 기능

### 설치 가이드 (3단계)
1. **BRAT** 설치로 베타 플러그인 추가
2. **Setup Wizard** — AI Provider, CLI 경로 설정
3. **GitHub Token 발급 & Repository 연결**
