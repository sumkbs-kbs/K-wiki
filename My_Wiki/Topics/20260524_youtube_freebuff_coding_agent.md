## 📚 주요 내용

### 1. FreeBuff란?
- 터미널(CLI)에서 실행되는 AI 코딩 에이전트
- **구독료제로, API 키 불필요** — 대신 텍스트 광고를 통해 무료 유지
- `npm install` 한 번으로 설치 시작 가능

### 2. 아키텍처: 9개 서브에이전트
- 파일 선택기, 코드 리뷰어, 브라우저 사용, 파일 언급, Bash 모드, 웹 리서치 등
- **9개의 전문화된 서브에이전트**가 각기 다른 작업 수행
- ChatGPT 딥 씽킹 통합 옵션

### 3. 지원 모델
- DeepSeek系列, MiniMax系列 등 다중 모델 지원
- 다른 모델이 다른 워크플로우 단계 담당
- 모델별 프라이버시 고려 필요 (DeepSeek는 학습 데이터 수집 가능)
