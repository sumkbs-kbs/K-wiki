# 이게 무료? 5분만에 내 목소리 복제해서 영상 자동화하기

## 요약
에딕초이 채널에서 공개한 영상. Claude Code 구독만으로 무료로, 한 마디도 녹음 안 하고 한 줄도 코드 안 짜고 나만의 목소리로 영상을 자동생성하는 워크플로우를 설명합니다. 핵심은 VoiceBox(로컬 음성복제) + HyperFrames(AI 영상생성)의 조합.

## 핵심 내용

### 2단계 워크플로우 (총 5분)
1. **VoiceBox** — 30초~1분 샘플만으로 내 목소리 복제. 로컬 실행으로 외부 서버 전송 없음(프라이버시 안전). GitHub 25,000+ Stars.
2. **HyperFrames** (by HeyGen) — "이런 영상 만들어줘" → HTML로 짜줌. 상업용 무료 라이선스. Claude Code에 한 줄 명령으로 스킬 설치.

### 핵심 메시지
- 두 도구 모두 Claude Code로 한번에 깔리고 영상까지 자동
- 결국 중요한 건 **디렉터의 안목**(도구 자체가 아니라 지시자의 시야)
