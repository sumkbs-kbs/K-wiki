# ⚡ 선박 BUS DUCT(부스덕트) — 전력을 위한 케이블 대안 시스템

> **분류:** 선박 전기 설계 핵심 부품
> **작성일:** 2026-05-26
> **관련 표준:** [[IEC 60092-201_System_Design]], [[IEC 60092-301_Generators_Motors]], [[IEC 60092-350_Cables]]

## 1. 부스덕트(Bus Duct)란?

**부스덕트(Busduct)**, 또는 **부스바 트렁킹(Busway Trunking)**이란, **전기를 전달하는 도체(구리/알루미늄 부스바)를 금속 외함 안에 일체형으로 보호하여 설계한 전력 배선 시스템**입니다. 기존 개별 케이블을 한데 묶어 쓰는 방식과 달리, **모든 상(phase)를 하나의 튜브 안에 담는 구조**로, 대용량 전력을 효율적으로 전송합니다.

> **비유:**  
> **전력 도로** — 한 개 도로(케이블)에 자동차(전류) 여러 대가 가는 대신, 여러 차선(부스바)을 하나의 터널(덕트)로 감싸는 구조입니다. 대용량 전류(수백~수천 암페어)를 효율적으로, 안전하게, 공간도 적게 차지하며 전달합니다.

## 2. 선박 부스덕트의 핵심 특징

### 2.1 기본 구조

| 구성 요소 | 설명 |
|--|--|
| **도체(Conductor)** | 구리(Cu) 또는 알루미늄(Al) 부스바. 3상/4선/5선 구성 |
| **절연체** | 도체를 전기적으로 격리하는 절연재. 방화·내열 등급 필수 |
| **외함(Jacket / Enclosure)** | 금속재질(강판/알루미늄 합금). 외부 충격·염해 방지 |
| **접속부(Expansion Joint)** | 열팽창·수축 허용을 위한 신축 조인트 |
| **분기박스(Tap-off Box)** | 하위 배전반으로 전력을 인출하는 플러그 인 박스 |

### 2.2 선박 환경 적합성

| 조건 | 대응 방식 |
|------|------|
| **염해(염분 부식)** | 외부 부식 방지를 위해 부식 방지 도장·도금 처리 (Zn, Al-Zn 합금) |
| **방화(Fireproof)** | 선박 내화 기준에 맞는 **방화 절연재** 사용 (UL 94 V-0 등) |
| **방진(Anti-vibration)** | 선박 엔진·프로펠러 진동에도 단선되지 않도록 **신축지주·탄성 지지대** |
| **밀폐성** | 침수·습기·먼지 차단 IP 등급 (IP54 이상 권장) |

## 3. 부스덕트 vs 케이블 — 선박 적용 비교

| 구분 | **부스덕트 **(Busduct) | **케이블 **(Cable) |
|------|------|------|
| **전류 용량** | **대용량 적합**: 200A ~ 6,000A+ | 소~중용량에 적합 |
| **설계 및 계획** | 소비 위치 고정, 전체 부하만 계획에 필요 | **유연성 높음** — 소비 위치 변동 가능 |
| **설치 공간** | **소형, 일체형** — 덕트 내부에 도체 일체 | 케이블 뭉치로 공간 많이 차지 |
| **설치 시간/비용** | 대용량 구간에서 **10~30% 절약** 가능 | 소용량 구간에 유리 |
| **단락 보호** | 외함이 기계적·전기적 보호 | 외부 충격 시 절연 파괴 위험 |
| **유지보수** | **간편** — 계통 단순화·접속점 적음 | 접속点多·검출 어려움 |
| **열방출** | 적음 (외함 차단) | 상대적으로 많음 |
| **중량** | 경량화 가능 (특히 Al 부스바) | 상대적으로 무거움 |
| **선급 승인** | 각 선급 규격에 따라 별도 승인 필요 | IEC 60092-350 기준에 따라 승인 |

## 4. 선박 부스덕트의 설계 기준

### 4.1 주요 설계 고려사항

1. **전압강하 계산** (Voltage Drop): 선미(Propulsion Motor 등)까지 전압강하지 규정치 이내

2. **단락전류 계산** (Short-Circuit Current): 선박 부스덕트의 최대 단락전류값 계산 및 보호기기 선정. ABS DC 전력계통 요구사항(I.3)에 따라 Maximum Calculated Short-Circuit Current 값 명시

3. **전원 선택적 차단** (Selective Coordination): IEC 60092-201의 선택적 보호 차단 원칙 준수

4. **절연 저항**: 절연재는 선박 내화 기준에 따라 인가전압, 온도, 습도 조건에서 안정적

5. **부하 분석** (Load Analysis): 부스덕트의 단면적과 전류 용량은 부하 계산 결과에 따라 선정

6. **선급 심사**: ABS, KR, DNV 등 선급 심사 통과 필요 — 설계도서 제출 → 검토 → 시공 검사 → 현장 시험

### 4.2 부스덕트 절연 등급

| 등급 | 설명 | 선박 적용 |
|------|------|------|
| **A급 **(절연체가 부식 방지형) | 도체·절연체 모두 부식 방지 처리. 부식 환경(선미, 엔진룸)에 적합 | **선박 엔진룸, 해수 근처 필수** |
| **B급 **(부식 방지형) | 외부 도체만 부식 방지 처리. 절연재는 일반형. 내부 공간에 적용 | 일반 배전 계통 |
| **F급 **(방화형) | 불연성·발화 지연 절연재 사용 (SOLAS/IMO 기준). 화재 안전구간 필수 | 선실, 항해실, 비상계통 |

## 5. 선박 부스덕트 설계 예시

### 5.1 설계 프로세스

1. **부하 분석** (Load Analysis)
   - 전체 부하량(kW/kVA) 계산
   - 각 지점 전류 용량(A)산정
   - 전압강하 검토

2. **도체(부스바截面) 선정**
   - 전류용량 > 최대 부하 전류 (여유도 1.25배 이상 권장)
   - 구리: 600~1,200A/mm² (환경적용)
   - 알루미늄: 400~800A/mm²

3. **외함(Enclosure) 선정**
   - 침수/습기/염해 환경 → IP54 이상
   - 방진·방화 등급 확인

4. **보호기기 선정** (Protection Devices)
   - 단락전류 계산 결과에 따라 차단기(Fuse/CB) 선정
   - 보호기기 조정(Protection Device Coordination Study) 수행

5. **선급 심사** (Class Society Review)
   - 설계도서 제출
   - 승인 후 시공

6. **시공 및 시험**
   - 밀폐성 시험 (IP 시험)
   - 절연저항 시험
   - 접지저항 시험
   - 가동 시험

### 5.2 설계 예시 (상세)

**예시: 일반화물선 (Deadweight 35,000톤) — Main Switchboard(MSB) → 각 분전반**

| 항목 | 수 |
|------|---|
| 전압 | 440V, 60Hz, 3상 |
| 부스덕트 전류 용량 | 2,500A (구리 부스바) |
| 절연 등급 | F급 (방화형) |
| 외함 IP 등급 | IP54 (방진·방수) |
| 외함 부식 처리 | Al-Zn 합금 도장 |
| 접지 부스바 | 단면 50x6mm 구리 |
| 선급 | 조선학회(KR) 심사 |

## 6. 부스덕트의 적용 단계 (Application Phases)

| 단계 | 내용 |
|------|------|
| **1. 예비 계획** | 전체 부하 분석, 부스덕트 전류 용량 산정 |
| **2. 상세 설계** | 도체 부면적, 외함, 절연 등급, 접속부, 분기박스 선정 |
| **3. 제조/납품** | 선급 승인 설계도서 기반 제조. 시운전·시험 |
| **4. 시공** | 선박 건조 중 또는 도크에서 시공. 절연·접지 검사 |
| **5. 가동** | 절연저항·단락전류 시험 → 전력 공급 |

## 7. 주요 장점과 단점

### 7.1 장점 (Advantages)

- **대용량 전송**: 2000A 이상 대용량 전력에 최적. 소용량 케이블 대용량 전류는 전압강하 및 열 문제 발생
- **설계/시공 효율**: 케이블 뭉치보다 **작고 가벼워 공간 및 설치비 절감** (10~30% 절약)
- **유지보수 간편**: 계통 단순화 — 단선·접점 불량 감소 — 검사 손쉬움
- **전력 공급 안정성**: 외함으로 보호됨 — 외부 충격·부식·침수·염분에도 강건
- **600V 이상의 전압 계통**에 적합.

**특히: 조선소의 Cable 전장 혁신 설계** — [[조선해운_Cable_전장_혁신_5%절감_전략]]의 핵심 요소로 활용 가능.

### 7.2 단점 (Limitations)

- **유연성 부족**: 소비 지점 변동 시 설계 변경이 케이블보다 어려움
- **비용**: 소용량 구간에서는 케이블보다 단가가 비싸짐
- **선급 승인**: 별도 설계도서·시행검사 → 심사 기간 추가 필요
- **특정 환경 적용 불가**: 극한 부식 환경에서는 특수 가공 필요 (단가 상승)

## 8. 부스덕트 종류 분류

### 8.1 절연 구조별

| 유형 | 특징 |
|------|------|
| **충진형**: 내부 부스바를 절연재로 충진 (단일체). 방수 방진 우수. |
| **공기형**: 도체 간 공기 절연. 방화·내열성 우수 (선박 내화 구간에 적합) |
| **외부절연형**: 외함·내부 부스바 분리. 유지보수 용이. |

### 8.2 전류 구조별

| 유형 | 특징 |
|------|------|
| **단상 3선** | 단일 상(Phase) → 중성점 포함. 소용량 |
| **3상 4선** | 3상 + 중성점. 일반 선박 계통 |
| **3상 5선** | 3상 + 중성선 + 접지선. 최신 선박 계통 |

### 8.3 외함 구조별

| 유형 | 특징 |
|------|------|
| **밀폐형** | 완전 밀폐 IP54 이상. 방진·방수·방염 |
| **통기형** | 약간의 통기구 있어 방열 용이 (저온 환경용) |

## 9. 선박 부스덕트 및 케이블 — 선택 기준

| 조건 | 권장 방식 |
|------|------|
| **대용량 계통** (≥600A) | **부스덕트 우선** |

| **엔진룸·선미 **(염해 환경) | **A급 부스덕트** |

| **선실 **(일반 계통) | 케이블·부스덕트 혼용 |
| **방산주 **(선급 심사 통과) | 부스덕트는 별도 심사 필요 |

## 11. 설계 및 실전 적용 참고사항

1. **전류 용량 여유**: 부스덕트 전류 용량 = 부하전류 × **1.25** (여유도)
   
2. **열팽창 대응**: 신축조인트(Expansion Joint) 필수. 선박의 열팽창(온도 60°C → -20°C) 대응.

3. **접지**: 접지 부스바 단면 50x6mm(구리). 접지 저항 **1Ω 이하** 유지 (선급 기준).

4. **연결 방식**: 플랜지 단자(Flanged End), 박스형(With Box) — 발전기·변압기 외함 연결에 사용

5. **선급 심사 제출물**:
   - 부스덕트 형식 검사서 (Type Test Report)
   - 절연 시험 결과서
   - 방화 등급 시험 보고서
   - 단면적·전류 용량 계산서

6. **Cable 전장 혁신 **(5% 절감) — [[조선해운_Cable_전장_혁신_5%절감_전략과 연계 — 부스덕트로 Main Switchboard(MSB) → 각 분전반 구간을 대체하면 Cable 작업 30~40% 절감 가능.
