## 2. 핵심 내용 구조화

### AI 엔지니어링의 핵심 레이어

```
["AI 엔지니어링 학습 로드맵"]
  → Layer 1: Python + 데이터 조작 (NumPy, Pandas)
  → Layer 2: 머신러닝 기초 (Scikit-learn)
      → Regression, Classification, Clustering
  → Layer 3: 딥러닝 핵심 (PyTorch)
      → 신경망, CNN, RNN, Attention
  → Layer 4: LLM / 생성 AI
      → Transformer, Fine-tuning, RAG, Agents
  → Layer 5: MLOps / 배포
      → 모델 서빙, CI/CD, 모니터링
```

### 학습 원칙
- **사전 지식 없음**을 전제로 한 단계별 접근
- 실제 프로젝트 기반 학습 강조
- 이론 → 구현 → 검증의 사이클 반복
