# Hermes AI Agent + Obsidian + Omi 완벽 메모리 시스템 (무료)

## 요약
Julian Goldie SEO 채널에서 공개한 영상. 세 가지 무료 도구(Hermes Agent, Obsidian, Omi)를 조합하여 영구 기억(Persistent Memory) 시스템을 구축하는 방법을 설명합니다. Omi가 배경에서 화면/마이크 활동을録화하여 일일 메모리를 생성하고, Obsidian Vault로 싱크하면 Hermes Agent가 그 메모리를 읽어 목표, 프로젝트, 일정, 선호도 등을 이해하여 맞춤형 자동화를 제안합니다.

## 핵심 내용

### 세 가지 무료 도구
1. **Hermes Agent:** AI 에이전트. Vault의 로컬 파일을 읽어서 컨텍스트를 이해하고 개인화된 대시보드/ 자동화 제안
2. **Obsidian:** 지식 베이스. 모든 메모리를 markdown 노트로 통합 관리
3. **Omi:** 백그라운드 녹화 도구. 화면과 마이크 활동을 자동으로 녹화하여 일일 메모리 생성

### "무한 컨텍스트 엔진"(Infinite Context Engine) 프레임워크
- Hermes에 메모리를 주입하여 각 세션마다 컨텍스트를 다시 설명할 필요 없음
- 시스템이 사용자의 목표, 프로젝트, 일정, 선호도를 자동으로 학습
- 자동화된 출력: 일일 다이제스트, 메모리 정리, 콘텐츠 아이디어 추출, 주간 성과 리포트

## 핵심 인사이트
1. **메모리 > 지능:** 메모리가 풍부한 Hermes가 단순한 대화형 LLM보다 훨씬 유용함
2. **완전 무료 도구 조합:** 세 도구 모두 무료/open-source
3. **자동 컨텍스트 수집:** Omi가 백그라운드에서 활동을 기록하므로 수동 개입 최소화
4. **일일/주간 루프 자동화:** 메모리 정리, 콘텐츠 추출, 성과 리포트 등 자동 사이클
5. **로컬 프라이버시:** 모든 데이터가 로컬 Vault에 저장, 외부 전송 없음
