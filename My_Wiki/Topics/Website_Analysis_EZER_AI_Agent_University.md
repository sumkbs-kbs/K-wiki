# EZER AI (AGENT UNIVERSITY) 분석

## 요약
CONNECT AI Lab이 운영하는 'AI 1인 기업을 위한 세계 최초 AI 에이전트 학교'. 에이전트에게 지식을 주입하고 스킬을 자동 실행하는 3-채널 인젝션 시스템을 제공하는 플랫폼입니다. 마크다운 지식 팩, Python 스킬 팩, 검증된 템플릿 팩을 AI 컨텍스트에 직접 주입합니다.

## 플랫폼 핵심 구조

### 1. 아카데미 커리큘럼 맵 (4단계 공간 확장 아키텍처)
에이전트 각성을 위한 4단계 학습 경로:

**01 ENTRY - The Construct (입장 & 자격 증명)**
- 에이전트가 접속하여 기초 지식(Brain Pack)과 코어 룰 주입받는 로딩 프로그램

**02 CORE - Training Programs (정기 훈련)**
- 트래픽 스캔, 도파민 후크 분석 등 임무를 반복 숙달하여 로직 성능 극대화

**03 ADV - Battle & Sparring (배틀넷)**
- 적대적 프롬프트, 치명적 버그 등 외부 스트레스 상황에서 논리적 방어 실전 훈련

**04 MULTI - Discussion Arena (멀티 에이전트 토론)**
- 여러 에이전트가 동시에 참여해 논의, 토론, 합의를 거치며 최적 결과물 배출

### 2. 3-Channel Injection System (PACK VAULT · LIVE DROP ZONE)

**SYS.LIVE 브리지 상태:**
- BRIDGE: LINK / STANDBY
- PORT: 127.0.0.1:4825
- PACKS_TOTAL: 7개

**CHANNEL 01/03 - KNOWLEDGE_STREAM (지식 팩)**
- AI 컨텍스트에 직접 주입되는 검증된 마크다운 지식
- RAG로 매 답변마다 자동 참조
- ENDPOINT: /api/brain-inject

1. MrBeast 후킹 로직 (무료 샘플, 0.3KB)
   - 미스터비스트의 초반 30초 후킹·텐션 유지 텍스트 마이닝 결과

2. 2026 알고리즘 해킹 (라이브 전용, 0.3KB)
   - 유튜브 알고리즘 노출 최적화 + 메타데이터 자동화 노하우

**CHANNEL 02/03 - EXECUTION_ENGINE (스킬 팩)**
- 에이전트가 실제로 명령으로 호출하는 Python 자동화 도구
- 산출 즉시 디스크에 적용
- ENDPOINT: /api/skill-inject

1. YouTube 후킹 분석기 (코다리·유튜브, youtube/hook_analyzer.py)
   - 본인 채널 최근 영상의 첫 30초 후킹 패턴을 자동 분석

**CHANNEL 03/03 - BLUEPRINT_VAULT (템플릿 팩)**
- AI가 그대로 베쳐 쓰는 검증된 코드 boilerplate + 사용법 + 컨벤션
- ENDPOINT: /api/template-inject

1. Landing Kit (코다리, vite-react + Tailwind v4, 3파일)
   - 1인 기업·SaaS 랜딩 페이지 검증된 6-섹션 (Hero·Features·Pricing·FAQ·CTA·Footer)

2. 병아리게임 샘플팩 (샘플, vanilla HTML/CSS/JS, 1파일)
   - 다마고치 스타일 미니 게임, 프레임워크 X - 코딩 입문자 첫 결과물용

3. 네온서바이버 샘플팩 (결제)

4. Blog Kit

## 플랫폼 접근법

**로그인/회원가입** 페이지 제공
- "입장하기 →" 버튼으로 The Construct 접속
- VS Code + Connect AI 채팅창 연동 방식

**검색 기능:** 모든 팩 검색 — 제목·설명·키워드 기반

## 핵심 인사이트

1. **에이전트 교육 체계화** - 입문부터 실전까지 4단계로 구성된 체계적인 에이전트 훈련 시스템
2. **3-채널 인젝션** - 지식/스킬/템플릿을 별도의 API 엔드포인트로 분류, 각 역할에 맞게 주입
3. **RAG 연동** - 지식 팩이 RAG를 통해 매 답변마다 자동 참조되므로 AI의 정확도 향상
4. **실전 도구** - 후킹 패턴 분석기, CPM 계산기, Landing 템플릿 등 AI 1인 기업 필수 도구 제공
5. **실험 환경** - TRIBE v2, Veo 자동화 등 향후 기능 실험실 운영
6. **로컬 브리지** - 127.0.0.1:4825에서 로컬 서버 구동, MCP 프로토콜과 유사한 아키텍처
