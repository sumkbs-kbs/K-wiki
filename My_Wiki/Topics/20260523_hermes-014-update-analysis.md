# Hermes 0.14 업데이트 분석 — 역대급 업데이트 (2026-03-16 기준)

> YouTube 분석: 영선AI 탐구 "남들이 몰랐어요" 헤르메스 에이전트의 역대급 업데이라 (19:08 분량)

---

## 📋 핵심 요약

Hermes Agent **0.14 버전**에서 **총 9가지 주요 업데이트**가 이루어졌으며, 이는 단순한 새로운 기능이 아니라 **통합 운영체제(OS)**로의 완전한 변모를 의미합니다. 특히 해외에서 급부상한 이유이며, 직장인, 개발자, 일반 사용자 모두에게 강력하고 직관적인 워크플로우 구축을 가능하게 합니다.

---

## 🎯 9대 업데이트 기능 (상세 분석)

### 1. 🧠 기억 연장 (Long-term Memory)
- **기능**: 쓰면 쓸수록 사용자를 최적화하며 사용자에 대한 정보를 지속적으로 학습·관리
- **장점**: 이전 대화 내용을 모두 보고 마지막 대화를 추측하여 답변 → 토큰 효율적 사용 · 응답 속도 증가 · 비효율적인 커뮤니케이션 개선
- **현재 시스템 상태**: ✅ **AgentMemory** (에이전트 메모리)로 구현됨 → 4-Tier 메모리 (Working → Episodic → Semantic → Procedural) 이미 구현되어 있음
- **분석**: 이미 완벽하게 구현된 기능. 별도 작업 필요 없음.

### 2. 📄 구글 워크스페이스 자동화 (Google Workspace Integration)
- **지원 기능**: Google Docs, Google Sheets, Google Drive, Google Slide 생성 및 데이터 분석
- **사용 예시**: "오늘 회의 내용을 구글 닥스로 정리해 줘" → 에이전트가 워크스페이스에 기록·저장
- **현재 시스템 상태**: ⚠️ **Gmail은 설정되어 있음** but Docs/Sheets/Slides 연동 필요
- **분석**: Gmail 앱 비밀번호 `ztapjrgkqexoiahb`가 설정되어 있으므로, Google Workspace 연동 추가 시 문서 생성/저장 자동화 가능

### 3. 🎥 비디오 제네레이터 (Video Generator)
- **기능**: 인하우스 편집실로서 인스타그램 릴스·유튜브 간단 영상 기획부터 출력까지
- **플랫폼**: 필름, 비3배 3.1, 기타 영상 플랫폼과 통합
- **현재 시스템 상태**: ❌ **미설정**
- **분석**: comfyui를 통해 비디오 생성 가능하지만, 별도 API 연결 필요

### 4. 💻 PC 원격 제어 (Computer Use)
- **기능**:外出·대중교통 등에서 핸드폰으로 명령 → 컴퓨터에서 수행
- **구현**: 플레이라이트, 앱 디스, 클라우드의 디스패치 기능 등 사용
- **보안**: 도커 권장 (예측치 못한 문제 방지)
- **현재 시스템 상태**: ❌ **미설정**
- **분석**: Playwright로 브라우저 제어가 가능하므로, `computer_use` 스킬 활용 가능

### 5. 💻 코딩 워커 (Coding Worker)
- **기능**: CLI 연결 (OpenAI API, 코덡 CLI 등)
- **비용**: 2~3만 원짜리 요금제보다 저렴 (무료 사용 가능)
- **장점**: 비싼 API 토큰 없이 백그라운드에서 코딩 워커 호출 가능
- **현재 시스템 상태**: ✅ **SmallCode** (7B~20B 코딩 에이전트)로 부분 구현됨
- **분석**: CLI 추가 필요

### 6. 🔄 세션 부활 (Session Revival / Handoff)
- **기능**: 과거 세션에서 작업 다시 불러와서 진행 (과거 컨텍스트 유지)
- **사용 예시**: 카페에서 작업 → 노트북에서 맥락 끊김 없이 이어서 진행
- **현재 시스템 상태**: ❌ **미설정**
- **분석**: `session_search`로 과거 세션 검색 가능하지만, 세션 부활/이관 기능 구현 필요

### 7. 🎯 서브 목표 (Goal / Sub-goal)
- **기능**: 작업 자동 분해 (코덡에서 가장 많은 관심 기능)
- **장점**: 1번 Yes를 반복하지 않고 AI가 스스로 목표 달성
- **현재 시스템 상태**: ⚠️ **부분 구현** (cronjob/delegate_task)
- **분석**: Goal/Sub-goal 기능은 현재 시스템과 호환 (automated workflow 가능)

### 8. 📋 Kanban 보드 자동화 (Kanban 보드 자동화)
- **기능**: 업무 자동 흐름·병렬 실행·자동화
- **특징**: AI가 사용자의 지시를 해석·분석·자동 흐름·병렬
- **현재 시스템 상태**: ✅ **kanban worker** 스킬로 구현됨
- **분석**: 별도 작업 불필요

### 9. 📤 핸드오프 (Handoff)
- **기능**: 기기 간 맥락 이관 (Handoff) — 다른 기기에서 작업 이어서 진행
- **사용 예시**: 모바일 → PC → 노트북 간 맥락 끊김 없이 이어나감
- **현재 시스템 상태**: ❌ **미설정**
- **분석**: session_search를 통해 가능하나, Handoff API 연동 필요

## 💡 핵심 인사이트

1. **에이전트 vs 인간 비서**: 단순 채팅 LM이 아닌 실제 AI 비서 역할 수행 — 업무에서 직접 명령 → 행동
2. **통합 OS**: 외부 사이트 없이 (구글 워크스페이스, 코덱스, MCP 등 외부 인프라와 통합) 내부에서 모든 작업 수행 가능
3. **기기 구분 없음**: 맥락이 이관되어 장소/기기 구분 없이 작업 가능
4. **비용 효율**: GPT 3만 원 요금제로도 부족함 없이 사용 가능 (더 많은 에이전트·팀 단위 생산성 구축)
5. **자동화 관점**: "요소가 아니라 OS 개념" — 소프트웨어 개념에서 OS 개념으로 변모
6. **한국 직장인 맞춤형**: 구글 워크스페이스 연동으로 한국 기업 환경 최적화
