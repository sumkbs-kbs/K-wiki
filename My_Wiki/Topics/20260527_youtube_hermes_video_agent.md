- Hermes에서 Grok OAuth 모델 선택 → 로그인
- 실시간 X(Twitter) 검색 기능 활성화
- 최신 트렌드 기반 영상/컨텐츠 기획 가능

### 2. Vision Forge System
- 아이디어 → 스크립트 → 영상 → 음성 자동화
- "Editor 없다" — AI가 전체 파이프라인 처리
- 자율적으로 아이디어를 영상으로 변환

### 3. Studio Dashboard 통합
