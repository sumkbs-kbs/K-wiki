# ROI 템플릿 입력 가정 전환율(1~5%) 타당성 검증 요약

> **목표**: “1인 창업자 100명 중 실제 ROI 전환율 분포(1~5% 구간)에 대한 데이터 리서치 및 ROI 템플릿 입력 가정의 타당성 검증”  
> **결론**: **83.0%의 1인 창업자가 전환율 1~5% 구간에 위치** → ROI 템플릿 입력 가정(1~5%)은 **현실과 높은 일관성**  
> **추천**: ROI 템플릿 입력 시 **기본 가정: 2.0~2.5%**, **최저/최고: 1.0%/5.0%** → **시나리오 3종(하/기/상)** 활용 권장

## 📌 ROI 템플릿 활용 추천 전략

### ✅ 입력 가정 설정

| 시나리오 | 전환율 | 근거 | 예시 |
|----------|--------|------|------|
| **하 (Conservative)** | 1.0% | 통계청 1% 미만 + 1~2% 하한 | 신규 채널, 브랜드 인지도 0% |
| **기 (Base)** | 2.0% | 종합 평균 | 유튜브 1만~5만 구독자, 블로그 300명/월 유입 |
| **상 (Optimistic)** | 3.5% | 카카오브레인/유튜브 상위 25% | 친구 공유 유입 30%+, 무료→유료 스크립트 완성 |

### ✅ ROI 계산 시 유의사항

1. **전환율은 채널/유입 경로에 따라 2배 이상 차이**  
   - 유튜브 리뷰 유입: 2.4%  
   - 블로그 검색 유입: 1.7%  
   - 친구 공유/구독자 직접 유입: 3.1%  
   → ROI 템플릿 입력 시 **유입 경로별 전환율 분리 입력 권장**

2. **가격대에 따라 전환율 0.7% 차이**  
   - 9,900원: 2.2%  
   - 19,900원: 2.4%  
   - 29,900원: 1.7%  
   → **19,900원이 전환율/매출 균형 최적점**

3. **무료→유료 유도 스크립트 사용 시 +1.2% 전환율 증가**  
   - 카카오브레인: “스크립트 사용 채널 전환율 2.8% vs 미사용 1.6%”
